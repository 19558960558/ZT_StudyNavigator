# ZT_学习导航

#### 介绍
ZT学习导航是一款专为应对B站首页推荐算法干扰而设计的效率工具。该工具旨在解决用户在学习过程中容易受平台推荐内容干扰的问题，通过提供纯净搜索通道直达B站搜索结果页，有效规避首页推荐内容带来的注意力分散。同时支持用户自主添加常用学习资源链接，构建个性化学习资源库，确保快速访问目标资源，从根本上杜绝无效浏览行为，切实保障专注力与学习效率的最优化。
